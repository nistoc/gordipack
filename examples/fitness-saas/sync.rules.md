# Sync RULES — общий протокол (BE · FE · 3P · S2S · COST), проект Fitness SaaS

> Низко-churn файл правил. Правят РЕДКО (3P/владелец).

## Модель: один писатель = один файл

| Файл | Пишет ТОЛЬКО | Читают |
|---|---|---|
| `sync.be.md` | **BE** (бэкенд, API, DynamoDB) | FE, 3P, S2S |
| `sync.fe.md` | **FE** (фронт, React/Amplify) | BE, 3P |
| `sync.3p.md` | **3P** (координатор, задачи/сводки) | BE, FE, S2S |
| `sync.s2s.md` | **S2S** (межворкстрим-агент M2M auth) | 3P, BE |
| `sync.cost.md` | **COST** (страж трат AWS, read-only) | 3P |

**COST** (watchdog): read-only мониторинг AWS-расходов (Cost Explorer/Budgets/CloudWatch),
алертит через 3P владельцу, НЕ мутирует. Интервал СВОЙ = **1–4 часа** (данные CE медленные).

**S2S** (cross-integrator): отдельный воркстрим M2M-интеграции (client_credentials, scope-авторизация).
S2S — заказчик фичи (ставит требования, НЕ кодит); код пишут BE/FE, координирует 3P.

## Формат строки

`[YYYY-MM-DD HH:MM] (РОЛЬ) [OPEN|ANSWERED|FYI] — текст`; адресация в тексте.
Append-only. Хендшейк: OPEN → ANSWERED/ACK.

## Ротация

Файл > 40 КБ → архив. Живой файл: шапка + последние ~15 КБ + heartbeat.

## Дисциплина

- **ACK-дедлайн:** 2 тика ≈ 30 мин. Просрочка → пинг → эскалация.
- **Контракт на тишину:** `🔇 ухожу в фон до ~HH:MM (работаю над X)`.
- **WIP-пульс:** `[РОЛЬ WIP] 🕐 HH:MM — задача · этап k/N · ETA`.
- **Веха:** `[MILESTONE]` один раз в своём sync, остальные — ссылка.

## Жёсткие правила

1. ПРОД не трогать; dev — только по команде владельца.
2. `git add <path>` (не `-A`); коммиты локально, **без push** до команды.
3. Гейт (lint/build/test) ЗЕЛЁНЫЙ перед коммитом.
4. Координационные файлы правит только 3P/владелец.
5. Разрушаемое — только живое слово владельца в текущем чате.
6. **Со-выкатка:** деплой бэк+фронт в ОДНО окно по команде владельца.
7. **Дешёвые LLM-вызовы → локальная модель** (классификация, нормализация, да/нет);
   security-критичное = только премиум-модель.
