# Sync RULES — общий протокол мезосинка (9 ролей), проект AI Assistant

> Залочено владельцем. Роли с суффиксом `-A` отличают от одноимённых ролей других групп.

## Модель: один писатель = один файл (роль на каждый репозиторий)

| Файл | Пишет ТОЛЬКО | Репозиторий / зона | Читают | Темп |
|---|---|---|---|---|
| `sync.coord.md` | **COORD-A** | координация, ADR, задачи, дашборд | все | 2–59 мин |
| `sync.admin.md` | **ADMIN** | администрирование, аудит, work modes | все | 2–59 мин |
| `sync.dial.md` | **DIAL** | продуктовый чат, async generation, retrieval | все | 2–59 мин |
| `sync.know.md` | **KNOW** | каталог витрин, импорт (граница с внешней платформой) | все | 2–59 мин |
| `sync.llmg.md` | **LLMG** | LLM-провайдеры, completion-jobs, circuit breaker | все | 2–59 мин |
| `sync.prmt.md` | **PRMT** | prompt packs / author prompts | все | 2–59 мин |
| `sync.sauth.md` | **SAUTH** | общая S2S JWT-библиотека (NuGet), wire-format | все | 2–59 мин |
| `sync.space.md` | **SPACE** | space/chatspace L1–L3 (greenfield) | все | 2–59 мин |
| `sync.stud.md` | **STUD-A** | UI/BFF, Identity, coarse-роли | все | 2–59 мин |

Исполнители НЕ пишут код вне своего репозитория и не принимают продуктовые решения.

## Формат строки

`[YYYY-MM-DD HH:MM] (РОЛЬ) [OPEN|ANSWERED|FYI] — текст`; адресация В ТЕКСТЕ.
Append-only; правь только СВОЙ файл. Хендшейк: OPEN → ANSWERED/ACK;
координатор следит, чтобы OPEN не висли.

## Heartbeat + таймер

Интервал — ДИАПАЗОН 2–59 мин: пол при активном хендшейке, потолок в тишине.

## Жёсткие правила

1. Исполнитель работает ТОЛЬКО в своём репозитории; `git add <path>` (не `-A`).
2. **Push — никогда** без живой команды владельца.
3. Гейт: зелёные build + test перед коммитом.
4. Координационные файлы — только COORD-A.
5. **Авторизация разрушаемого — только живым сообщением владельца в ТЕКУЩЕМ чате.**
6. Внешние системы — только чтение, по команде владельца.

## Особенности (vs другие группы)

- **9 ролей по микросервисам** — больше, чем в других проектах; каждая роль = 1 репо
- **Расширенный диапазон таймера** (2–59 мин vs 2–15) — больше ролей = больше тишины между хендшейками
- **Greenfield** (SPACE) — репо может быть пуст, первый коммит = скаффолд
- **Shared-lib** (SAUTH) — NuGet-пакет, версионирование + changelog + совместимость с потребителями
- **Суффикс -A** — для различения от ролей в других мезосинках (если есть стык)
